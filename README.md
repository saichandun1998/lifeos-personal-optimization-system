# LifeOS: Personal Optimization System

A high-performance dashboard for managing micro-habits, energy-aligned scheduling, life balance tracking, and quick cognitive capture.

## Key Features
- **Energy-Sync Scheduler**: Visualizes your daily energy flux and helps schedule high-impact tasks during peak performance windows.
- **Micro-Habit Loops**: Track tiny wins with automated streaks.
- **Life Equilibrium**: Radar visualization for balancing the six core dimensions of a flourishing life.
- **Quick Sparks**: A cognitive inbox for rapid thought capture.
- **Decision Dice**: Overcome analysis paralysis with probabilistic decision-making.

Built with React 19, Tailwind CSS, and HTML5 Canvas.
